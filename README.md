## VPS 是什么：一句话说清楚

VPS（Virtual Private Server，虚拟专用服务器）就是一台永远开着的远程电脑，放在某个数据中心里，24 小时联网，你可以用 SSH 远程进去，像操控自己的电脑一样操控它。

和"虚拟主机"的区别：虚拟主机是合租房，大家挤在一台服务器里共享资源，便宜但自由度极低。VPS 是自己租了一套小套间，有独立资源，可以随意折腾，装什么软件都行。

搞清楚自己属于哪一类，后面选套餐就不会乱。

## VPS 新手入门：买完之后该干什么

拿到 IP 和密码，很多人第一反应是"然后呢"。一步一步来。

### 第一步：用 SSH 连进去

Windows 用户装个 [Termius](https://termius.com) 或者直接用 PowerShell；Mac 用户打开终端直接输入：

bash
ssh root@你的IP地址


回车后输密码（输密码的时候界面不显示字符，直接输然后回车就行）。

第一次进去建议做几件事：

1. 改 SSH 默认端口（22 很容易被扫描，改成 10000 以上的随机端口）
2. 改 root 密码或者设置 SSH 密钥登录
3. 更新系统软件包

bash
apt update && apt upgrade -y   # Debian/Ubuntu
yum update -y                  # CentOS


### 第二步：选操作系统

新手强烈推荐 **Debian 12** 或 **Ubuntu 22.04 LTS**，教程多，出问题好查。CentOS 的生命周期已经结束，现在不建议选。

### 第三步：装宝塔面板（新手友好）

不会命令行也没关系。宝塔面板把环境部署、网站管理都做成了图形界面，安装之后浏览器打开 IP:8888 就能操作，和用虚拟主机差不多。

安装命令（Debian/Ubuntu）：

bash
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && bash install.sh ed8484bec


装完会给你面板地址和默认账号密码，保存好。

### 第四步：部署网站

装完宝塔，在面板里：

1. 一键安装 LNMP 环境（Nginx + MySQL + PHP）
2. 点"网站"→"添加站点"，填入域名
3. 上传网站文件，或者用宝塔里的"一键部署"直接装 WordPress

绑定域名记得先把域名 DNS 解析到你的 VPS IP，等个几分钟就生效。

## 流量用完了会怎样

这是很多新手担心的事。

DMIT 的处理方式是**降速不断网**。流量超出后不会直接停机，而是把带宽限速到 50Mbps-100Mbps（视套餐而定），网站还能正常访问，只是速度慢一些，等下个月流量重置就恢复了。

对比那些流量超了直接封机器的商家，这种处理方式算是友好的。

## 新手常见问题 FAQ

**Q：完全不懂 Linux，可以用 VPS 吗？**

可以。装上宝塔面板之后，绝大多数操作都是图形界面，和操作普通网站后台差不多。最基础的 SSH 连接学半小时就会了，不是门槛。

**Q：VPS 建站需要备案吗？**

国内服务器（阿里云、腾讯云等）需要备案，海外服务器（香港、日本、美国节点）不需要备案直接上线，这也是很多人选择海外 VPS 建站的原因之一。

**Q：DMIT 的 CN2 GIA 和 CMIN2 有啥区别，选哪个？**

CN2 GIA 是电信的顶级精品线路，三网回程都走 CN2 GIA，延迟更稳、丢包更少，价格也更贵。CMIN2 是移动的优化线路，移动用户体验非常好，电信联通也不差，但整体质量比 CN2 GIA 略低一档，价格便宜一些。

**对移动用户来说**，CMIN2 的体感有时不比 CN2 GIA 差多少；**对电信用户来说**，CN2 GIA 更稳。不确定可以先用测试 IP 分别 ping 一下，数据说话。

**Q：DMIT 的优惠码怎么用？**

在购物车页面，结算前有"优惠码"输入框，把对应的官方促销码填进去点"应用"，折扣自动计算。注意多数优惠码只适用于特定机房、特定套餐、且需要季付或年付才能触发，买之前对照清楚。

**Q：VPS 的 IP 被墙了怎么办？**

DMIT 支持每 15 天免费换一次 IP。换完测试一下新 IP 是否畅通，再继续用。15 天之内额外换 IP 收费 $5 一次，不算便宜，所以尽量用官方推荐的测试 IP 先确认 IP 质量再下单。
